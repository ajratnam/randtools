# RandTools

Common objects and functions, that I use in **my** various other projects.


## Installation

Via pip:

```bash
pip install randtools
```

Via Source (Poetry):

```bash
poetry build
pip install dists/randtools-1.0.0-py3-none-any.whl
```

## Requirements
Python 3 is required

## For Devs
All the test can be run via `pytest tests`
